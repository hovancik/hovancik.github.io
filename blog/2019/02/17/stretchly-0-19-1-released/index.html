stretchly 0.19.1 released

Posted under stretchly on Feb 17 2019

Hi there guys, I am really happy to release version 0.19.1 of stretchly. You can download it from downloads page.

stretchly is cross-platform open source app that reminds you to take breaks when working with computer.

If you enjoy stretchly, consider becoming patron at Patreon.

Fixed

  • fullscreen mode on Windows
  • ability to hide break ideas from break windows

Added

  • more debug info (chrome, electron, nodejs versions)
  • allow user to disable ability to postpone breaks

Changed

  • Simplified Chinese translations updated
  • Czech translations updated
  • Slovak translations updated

Thanks to everyone for support on Patreon and code contributions!
